## 二、需小程序管理员登录 [微信公众平台](https://mp.weixin.qq.com)（约 15 分钟）

### 1. 服务器域名

路径：**开发管理 → 开发设置 → 服务器域名 → request 合法域名**

添加（**两个都要**）：

```text
https://feijihe.top
https://guanli.feijihe.top
```

要求：

- 必须 **HTTPS**
- 证书有效（现网已满足）
- 不要带路径，不要带端口

### 2. 体验版

1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. **导入项目** → 选 `D:\Desktop\quote-weapp`
3. 确认 AppID 为 `wxa1d9f876327af0c0`
4. 本地预览：可勾选「不校验合法域名」
5. **上传** → 填写版本号如 `1.0.0`、备注「内部报价体验版」
6. 公众平台 **管理 → 版本管理 → 开发版本 → 选为体验版**

### 3. 体验成员

路径：**成员管理 → 体验成员** → 添加员工微信号

只有体验成员能扫体验版二维码，不对外公开。

## 四、常见问题

| 问题 | 处理 |
|------|------|
| request 合法域名校验失败 | 后台是否加了 `https://feijihe.top`；真机必须配域名 |
| 加载报价配置失败 | 应用机 3002 是否正常；`curl https://feijihe.top/api/quote_data` |
| 403 / 证书错误 | 检查 feijihe.top SSL |
