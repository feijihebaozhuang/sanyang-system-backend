# 商家编码 ↔ 三羊生产映射表

> **用途**：快麦商品「商家编码」与三羊刀模/规格的唯一关联依据（对齐阶段维护，**先填表再开发库存/损耗**）。  
> **主数据**：商品档案在 **快麦** 维护；三羊 **不复制** 商品标题做主键。  
> **规格展示**：订单侧以 **买家下单 SKU 属性** 为准（见锚点 `v8.33` 约定）。

---

## 维护责任

| 角色 | 职责 |
|------|------|
| 快麦维护人（C / 运营） | 在快麦创建/修改商品，保证 **商家编码** 准确、唯一（按规则） |
| 三羊维护人（生产/技术） | 在本表维护编码与刀模、产品类型的对应；编码变更时同步更新 |
| 客服 | 发现「有单无映射」「规格对不上」时登记异常列 |

**编码命名建议（会上可改）**：

- 格式示例：`FH-18-14-5-WJ`（类型-长-宽-高-材质缩写）或沿用现有厂内编码  
- 定制链接类 SKU：单独前缀或 `DZ-` + 客户/款号，避免与现货混码

## 与快麦字段对应（技术对照）

| 快麦侧 | 三羊侧使用方式 |
|--------|----------------|
| 商家编码 / `outerId` / `sysOuterId` | 本表「商家编码」列，关联刀模 |
| 商品标题 / `sysTitle` | **仅客服展示参考，不参与** 生产规格解析 |
| 订单子单 `skuPropertiesName` 等 | 生产规格、分类、定制判定（已实现） |
