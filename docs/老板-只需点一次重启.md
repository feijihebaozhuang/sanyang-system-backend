## 你只做 1 件事（阿里云网页，约 2 分钟）

### 情况 A：C 说「代码已 push，请部署」——用 **云助手**（不用 Workbench SSH）

1. 阿里云 ECS → **8.166.132.87** → **云助手** → **发送命令**
2. 粘贴下面整段（把 `你的Gitee令牌` 换成你在 Gitee 生成的私人令牌；需先 **Gitee 加你为仓库开发者**）：

```bash
export GITEE_TOKEN='你的Gitee令牌'
TMP=$(mktemp -d)
curl -fsSL "https://gitee.com/api/v5/repos/feijihesanyan/sanyang-system/tarball/main?access_token=${GITEE_TOKEN}" -o "$TMP/repo.tar.gz"
tar -xzf "$TMP/repo.tar.gz" -C "$TMP"
SOURCE_DIR=$(find "$TMP" -mindepth 1 -maxdepth 1 -type d | head -1)
export SOURCE_DIR
bash "$SOURCE_DIR/scripts/ops/bootstrap_repo_87.sh"
sudo bash "$SOURCE_DIR/scripts/ops/unlock_hermes_87.sh"
```

3. 等执行成功 → 飞书 @小马哥：**「部署好了，试一下」**
