# 客户/客服下单小程序

API 根地址：**https://guanli.feijihe.top**（3003）

## 项目路径

| 小程序 | 目录 | 用途 |
|--------|------|------|
| 客户下单 | `weapp/customer-order` | 选分类 → 填规格 → 算价 → 下单 → 查进度 |
| 客服接单 | `weapp/cs-order` | **微信登录**（与日常工作微信同号）→ 审核名下订单 → 一键联系客户 |

## 微信后台

**开发管理 → 服务器域名 → request 合法域名** 添加：

```text
https://guanli.feijihe.top
```

算价走 3003 代理本机 3001，无需再加 feijihe.top。

## 服务器 .env（87 stable）

```env
WX_MP_APPID=小程序AppID
WX_MP_SECRET=小程序AppSecret
MP_QUOTE_PROXY_BASE=http://127.0.0.1:3001
```

未配 AppSecret 时开发工具可用 dev openid 联调。

## 导入开发者工具

1. 微信开发者工具 → 导入 → 选 `weapp/customer-order` 或 `weapp/cs-order`
2. 本地调试可勾选「不校验合法域名」
3. 上传体验版前必须配置合法域名

## 客服账号

- 客服小程序须用**日常工作微信**登录（首次绑定 3001 账号后自动记住）。
- 3003 后台「客服员工」填**微信同号（手机号）**，与客户归属对应。
- 审单时可拨打电话 / 复制号码，到微信搜索添加客户。
- MySQL `users` 表角色为客服/主管/管理员（与 zean.feijihe.top 相同），仅用于首次绑定。

## 部署

后端随 3003 部署：`git pull && bash deploy.sh`（含 `mp_api.py`、`mp_auth.py`）。
