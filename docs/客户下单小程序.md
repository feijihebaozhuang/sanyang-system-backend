## quote-weapp 审单入口



首页底部 → **📋 客户下单审单（客服登录）** → 微信登录 → 待审核列表 → 详情（底部 **微信联系客户** / 拨打电话）



### 审单页「微信联系客户」说明



- **不是**加好友私聊：微信小程序**不能**直接打开与某个用户的微信好友聊天窗口。
- 点击后系统向该客户（客户下单小程序 openid）发送**小程序客服消息**；客户在微信里**直接回复**即可。
- 客服在手机微信打开 **「小程序客服助手」**（或公众平台客服工作台）接待对话。
- 微信限制：客户需曾点过客户小程序「联系客服」，或 48 小时内有过客服消息，否则无法主动发起；此时请用**电话**或**复制手机号**添加微信。
