### 🔴 2. 确认 213 无业务进程（2 分钟）


### 🟡 3. 213 每日 MySQL 备份（10 分钟）

213 上：

```bash
mkdir -p /home/admin/backup/sanyang /home/admin/scripts
cp /path/to/repo/scripts/ops/mysql_daily_backup_213.sh /home/admin/scripts/
chmod +x /home/admin/scripts/mysql_daily_backup_213.sh
# crontab -e 加一行（密码用 ~/.my.cnf）:
# 0 3 * * * MYSQL_PWD='...' /home/admin/scripts/mysql_daily_backup_213.sh >> /home/admin/backup/backup.log 2>&1
```

---

### 🟡 4. 156 权限分离（30～45 分钟）

**4a** 应用机生成 Token：

```bash
bash /www/feijihe/repo/scripts/ops/gen_vault_token.sh
# 记下输出，下面叫 YOUR_TOKEN
```

**4b** 应用机导出 JSON 到 156：

```bash
bash /www/feijihe/repo/scripts/ops/export_permission_to_156.sh
```

**4c** 156 安装 vault（先把 repo 同步到 156 或 scp 脚本）：

```bash
sudo VAULT_TOKEN='YOUR_TOKEN' APP_SERVER_IP='8.166.132.87' \
  bash /www/feijihe/repo/scripts/ops/install_permission_vault_156.sh
```

**4d** 应用机切换只读（156 内网 IP 替换 `<156内网IP>`）：

```bash
VAULT_URL='http://<156内网IP>:9443/permission_data.json' \
VAULT_TOKEN='YOUR_TOKEN' \
bash /www/feijihe/repo/scripts/ops/enable_app_vault_readonly.sh
```

验收：生产端改工序 → **403 已锁定**。
